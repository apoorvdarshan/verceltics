§ Setup — Three minutes, once

No account.
No magic link.

01

Create a token.

Generate a read-only personal access token in your Vercel account settings. Takes ten seconds.

→ vercel.com/account/tokens

02

Paste it once.

Open Verceltics, paste the token. We store it in the iOS Keychain — encrypted, app-scoped, sandboxed. Never crosses our servers because there are none.

→ iOS Keychain

03

Check anytime.

Open the app from anywhere. Pull to refresh. Tap a project. The live data is right there, on the device you already carry.

→ Done
